# Installation

This page explains how to install Parley and verify it loads.

## Steps
1) Copy the `parley` folder into `Packages/`.
2) Ensure `Package.toml` is present in the package root.
3) Start the server and confirm Parley loads.

## Folder Structure
```
Packages/
  parley/
    Package.toml
    Shared/
    Server/
    Client/
```
