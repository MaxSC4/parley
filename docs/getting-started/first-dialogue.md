# First Dialogue

This page walks through a small dialogue line by line.

## Example
```
start:
Narrator: Welcome.
- Continue. -> end
```

## Explanation
- `start:` defines the entry label.
- `Narrator:` is a speaker line.
- `-` defines a choice.
